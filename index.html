What You'll Build¶

  • A deployed agent running 24/7 on your own VPS infrastructure
  • A documented security architecture with sandbox isolation and identity controls
  • A domain-specific sub-agent with custom identity, skills, and workflow logic
  • A public GitHub portfolio showcasing your methodology and agent design

This is not a course about AI concepts.
It is a build environment for people who want working systems, not just familiarity with tools.

Why This Is Different¶

Not a course. An incubator.

You are not watching videos. You are building infrastructure, publishing outputs, and learning by shipping.

Invitation-only, referral-aware

Admission is curated. The workshop is the gateway into a deeper membership layer and the path toward founding status.

Portfolio as proof

Participants leave with a live agent, documented method, and a public GitHub Pages site that shows what they built.

Founding cohort path

The strongest early builders can earn their place in the first 31 founding members, with lifetime access and founding-only privileges.
